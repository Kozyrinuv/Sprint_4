# Sprint_4

Проект проверен в 2-х браузерах Chrome и Firefox.

В проекте реализованы 2 тестовых задания:
1- автотест проверки текстов "Выпадающего списока в разделе «Вопросы о важном» (файл CheckList.java )
2- автотест  проверки "Всего флоу позитивного сценария" (файл Check.Flow.java)
   в данном в соответствии с заданием реализованы 2 теста, один начинается с нажатия кнопки "Заказать"
   в правом верхнем углу главной страницы, второй тест аналогичен первому, но начинается с нажатия
   кнопки "Заказать" расположенной внизу  страницы в центре.
   Оба теста ФЛОУ реализованы с 2-мя наборами входных данных.

 В программе создан отдельный пакет PageObjects для 3-х страниц сайта:
 - HomePageScooter - главная страница сайта
 - PersonPage - страница заполнения данных о Заказчике
 - RentPage  - страница заполнения данных об условиях аренды

!!!!Тест ФЛОУ в браузере Chrome доходит только до формы "Хотите офрмить заказ? ДА - НЕТ
    Кнопка ДА на данной форме в браузере Chrome не нажимается и тесты на этом этапе падают
    В браузере Firefox тесты отрабатыват до конца (до формы отображения статуса заказа)

    Настройка на соответствующий браузер выполняется в файлах запускающих тесты 
    Check.Flow.java и CheckList.java
